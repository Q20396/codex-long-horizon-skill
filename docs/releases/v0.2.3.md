# v0.2.3

Release date: 2026-07-24

## Summary

v0.2.3 is a documentation and release-metadata maintenance release for the
Codex plugin package. It makes the stable repository marketplace installation
reference explicit for both bundled skills.

## Included Skills

- `long-horizon-engineering`
- `ai-video-production`

## What Changed

- Updated customer-facing installation documentation to recommend the immutable
  `v0.2.3` release reference for the Codex plugin marketplace.
- Retained `--ref main` documentation for users who intentionally want to
  follow unreleased repository state.
- Aligned plugin metadata, release manifests, package validation, CI, catalog
  validation, and release-readiness checks with v0.2.3.

## Safety Model

- Marketplace registration does not automatically install, connect, or grant
  external-system access to a skill.
- Skill updates remain check-only by default and require explicit approval
  before replacement.
- The release does not add dependencies, runtime behavior, automatic updates,
  or external-service integrations.

## Stable Installation

```bash
codex plugin marketplace add Q20396/codex-long-horizon-skill --ref v0.2.3
```

Use `--ref main` only when you explicitly want unreleased repository state.

## Validation

- Plugin package validation
- Release-readiness validation
- Fresh-install validation without modifying a real Codex installation
- Full local skill validation suite
