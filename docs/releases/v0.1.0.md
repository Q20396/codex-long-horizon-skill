# Codex Long Horizon Skills v0.1.0

Release: v0.1.0
Date: 2026-06-18

## Highlights

- Bundles the `long-horizon-engineering` skill for evidence-backed repository
  exploration, planning, implementation, validation, PR handoff, resumable
  work, and safety-aware logs.
- Bundles the `ai-video-production` skill for video briefs, scripts,
  storyboards, shot lists, visual prompts, asset manifests, preview plans, and
  render handoffs.
- Defines privacy and safety boundaries for secrets, client data, sensitive
  material, external tools, production actions, persistent memory, and public
  release artifacts.
- Provides a prompt, template, and example library for engineering workflows,
  research-backed writing, design reviews, presentations, notebooks, video
  planning, and reusable handoffs.
- Adds deterministic routing validation for bundled skills, expected trigger
  fixtures, skill description audits, safety checks, and release-readiness
  tooling.
- Packages the repository as a Codex plugin with `.codex-plugin/plugin.json`
  and a Git-backed repository marketplace at `.agents/plugins/marketplace.json`.
- Includes contributor and maintainer scaffolding: installation docs, upgrade
  guidance, issue templates, pull request template, security policy, code of
  conduct, release checklist, and first-contribution guide.

## Installation

Plugin marketplace installation was verified through an isolated Codex CLI
0.141.0 gate using the Git-backed repository marketplace:

```bash
codex plugin marketplace add Q20396/codex-long-horizon-skill --ref main
codex plugin add codex-long-horizon-skill@codex-long-horizon-skills
```
