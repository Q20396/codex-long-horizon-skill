## Highlights

- Long-horizon engineering skill for repository exploration, multi-file
  implementation, debugging, migrations, validation, PR handoff, and resumable
  work.
- AI video production skill for video briefs, scripts, storyboards, shot lists,
  visual prompts, asset manifests, preview plans, and render handoffs.
- Safety and privacy boundaries for secrets, client data, sensitive materials,
  external tools, production actions, and persistent logs.
- Prompt library, reusable templates, and examples for common workflows.
- Validation tooling for package structure, routing fixtures, safety checks,
  plugin packaging, fresh isolated installs, and release readiness.
- Open-source contributor scaffolding, including issue templates, PR template,
  community skills registry, security policy, code of conduct, and first
  contribution guide.
- Codex plugin packaging with `.codex-plugin/plugin.json`.
- Repository marketplace at `.agents/plugins/marketplace.json`.
- Installation verification for direct skill installs and deterministic plugin
  path checks.
