# v0.2.4

Release date: 2026-07-24

## Summary

v0.2.4 adds a proposal-only renderer runtime sandbox boundary for
AI-assisted video production.

## Included Skills

- `long-horizon-engineering`
- `ai-video-production`

## What Changed

- Added a renderer runtime sandbox protocol for local renderer assessment.
- Added an approval card that separates environment inspection, dependency
  installation, preview rendering, final rendering, external processing, and
  sharing.
- Clarified that renderer selection does not authorize installation, rendering,
  provider use, uploads, or sharing.
- Added local contract coverage for the sandbox boundary.

## Safety Model

- The protocol is proposal-only by default.
- No renderer, provider, browser automation, container, or external service is
  installed, started, connected, or invoked automatically.
- No automatic local-to-cloud fallback is permitted.
- Every state-changing step requires separate user approval.
