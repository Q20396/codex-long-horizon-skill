# Trigger Review

Use this template for a proposal-only review of explicit user-supplied,
aggregated skill usage observations.

Do not include raw prompts, conversation contents, Codex logs, shell history,
browser history, hidden-file contents, repository source contents, credentials,
tokens, API keys, user identities, client identities, account identifiers,
private absolute paths, GPS or location data, or connected-service data.

Completing this template does not change trigger descriptions, trigger
fixtures, routing metadata, lifecycle state, or installed skills.
No usage data was collected automatically.
