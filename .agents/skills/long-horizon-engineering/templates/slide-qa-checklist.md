# Slide QA Checklist

## Content
- [ ] Each slide has one main message.
- [ ] Claims are supported by evidence.
- [ ] No unsupported superlatives.
- [ ] Audience and decision are clear.

## Design
- [ ] Typography is consistent.
- [ ] Colors are consistent.
- [ ] Alignment and spacing are clean.
- [ ] Charts are readable.

## Data
- [ ] Units are shown.
- [ ] Dates are shown.
- [ ] Sources are shown.
- [ ] Calculations were checked.

## Export
- [ ] File opens successfully.
- [ ] Fonts render correctly.
- [ ] Images are not blurry.
- [ ] Speaker notes are present if requested.
