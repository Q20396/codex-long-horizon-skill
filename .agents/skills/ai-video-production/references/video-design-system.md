# Video Design System

Use this reference when a video project needs consistent visual direction across
scenes, thumbnails, generated images, animations, or render handoffs.

This is a design source of truth, not a brand clone. Use concrete style tokens
as inspiration for original work. Do not copy proprietary brand systems,
copyrighted assets, private client design files, or external project prose.

## Design Source Of Truth

For larger projects, maintain a lightweight `DESIGN.md` or design-system
handoff that includes:

- Audience and mood
- Typography direction
- Color tokens
- Spacing and layout rhythm
- Motion language
- Component patterns
- Responsive behavior
- Rationale for important choices

## Concrete Style Tokens

Define:

- Fonts or font categories
- Primary, secondary, accent, background, and warning colors
- Type scale
- Grid or safe-area rules
- Scene transition rules
- Caption style
- Thumbnail treatment
- Icon, mockup, or illustration style

## QA

- Visuals support the script.
- Captions are readable on mobile.
- Color contrast is adequate.
- Prompted assets match the design system.
- Generated images or clips do not imply false documentary evidence.
- Private or copyrighted materials are excluded unless explicitly approved.
