# Production Handoff Style

Use this optional style when a video plan needs to be handed to an editor,
motion designer, or render operator.

- Make script, storyboard, shot list, asset manifest, and render handoff align.
- Use placeholders for unapproved assets.
- Name approval status and unresolved risks.
- Keep licensing and privacy checks visible.
- Do not include generated media, external code, or private source content.
