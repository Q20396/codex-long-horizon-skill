# Short-Form Cinematic Style

Use this optional style for vertical social videos, launch explainers, and
cinematic short-form planning.

- Start from the central idea and audience.
- Use visual beats, motion, pacing, and atmosphere.
- Keep assets generic unless the user approves specific materials.
- Separate factual claims from speculation.
- Require approval before rendering, uploading, publishing, or using private
  assets.
