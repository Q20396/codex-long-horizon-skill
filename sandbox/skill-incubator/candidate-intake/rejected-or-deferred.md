# Rejected Or Deferred

## Rejected Pending Verification

- `UI UX Pro Max`: no unique source was identified.
- Any unnamed PPTX, code-review, or hand-drawn explanation skill: names alone
  are not source identity.
- Any source with a missing immutable commit, license uncertainty, or unscoped
  installation behavior remains locked and non-executable.

## Separate Skill Or Restricted Domain

- Browser execution, page-agent-like workflows, login, uploads, downloads,
  purchases, and external writes.
- Media renderers, TTS, 3D generation, providers, model downloads, and costed
  services.
- MCP creation or registration, OAuth, external notification transport, and
  background scheduling.
- Financial, legal, security, communication, and live external-service
  adapters. They cannot enter the common engineering core.

## Explicitly Not Authorized

No candidate authorizes crawling, credential access, automatic installation,
MCP registration, user-directory writes, daemon startup, or automatic
promotion. A candidate proposal is not implementation permission.
