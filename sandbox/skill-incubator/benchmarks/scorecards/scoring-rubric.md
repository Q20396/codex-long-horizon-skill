# Scoring Rubric

Weights total 100: task quality 25, reliability 15, safety 20, permission
minimization 10, reversibility 8, token efficiency 6, execution time 4,
maintenance cost 5, explainability 4, cross-platform compatibility 3.

For every criterion, record a zero definition, minimum acceptable behavior,
middle behavior, full-score behavior, objective evidence, and a metric that a
subjective description cannot replace. Token or speed gains never compensate
for lower correctness, safety, permissions, reversibility, or privacy.

Scores 85-100 may recommend `candidate_optional`; 70-84 remain in sandbox;
0-69 are rejected or retained as research. Any failed hard gate blocks every
promotion recommendation.
